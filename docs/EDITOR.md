# Редактор изображений и видео — Руководство пользователя

Вкладка **Редактор** — это полноценный нелинейный видеоредактор прямо в браузере. Здесь можно:
- Собрать видеоролик из изображений и видеоклипов
- Добавить переходы между клипами
- Наложить визуальные эффекты на каждый клип
- Добавить субтитры с анимацией
- Добавить несколько аудиодорожек со звуковыми эффектами
- Добавить картинку в картинке (PIP)
- Экспортировать итоговое видео в любом популярном формате

## Пошаговое руководство

### Шаг 1. Создание нового проекта

1. Перейдите на вкладку **Редактор**
2. Нажмите **Новый проект** (если хотите начать с чистого листа)
3. Введите имя проекта в поле в верхней части экрана

### Шаг 2. Добавление медиафайлов

Кнопки в панели инструментов:
- **+ Изображения** — выберите один или несколько файлов JPG / PNG / WebP / BMP
- **+ Видео** — выберите видеоклип MP4 / MOV / MKV / WebM / AVI
- **+ Аудио** — выберите аудиофайл MP3 / WAV / AAC / FLAC / OGG

Можно также **перетащить файлы** прямо в область предпросмотра — редактор определит тип по расширению.

После добавления изображения или видео появится на **видеодорожке** таймлайна. Аудио добавится в первую свободную **аудиодорожку**.

### Шаг 3. Настройка длительности изображений

- Для каждого изображения длительность задаётся в панели свойств (вкладка **Слайд** → поле «Длительность»)
- Чтобы задать одинаковую длительность всем изображениям сразу: введите значение в поле **«Длит. по умолч.»** вверху и нажмите **«Применить ко всем»**
- Длительность видеоклипа определяется автоматически; можно изменить в свойствах клипа

### Шаг 4. Расстановка клипов

- **Перетащите** клип за его тело, чтобы изменить порядок
- **Потяните правый край** клипа для изменения длительности
- **Shift + клик** — добавить клип к выделению
- **Ctrl + A** — выделить все клипы
- Клипы магнитно **прилипают** к соседним клипам, аудиодорожкам и позиции воспроизведения

### Шаг 5. Добавление переходов

1. Выберите клип (нажав на него)
2. В панели свойств перейдите на вкладку **Переход**
3. Выберите тип перехода из выпадающего списка
4. Задайте длительность перехода (рекомендуется 0.3–1.5 с)

Переход применяется к **началу** выбранного клипа (т.е. к переходу от предыдущего к данному).

### Шаг 6. Добавление визуальных эффектов

1. Выберите клип
2. Перейдите на вкладку **Эффекты** в панели свойств
3. Настройте ползунки эффектов или включите/выключите тумблеры

Изменения мгновенно отображаются в области предпросмотра.

### Шаг 7. Настройка эффектов входа/выхода клипа

В панели свойств на вкладке **Слайд** можно задать:
- **Эффект входа** — анимация появления клипа (Fade In, Zoom In, Slide from Left и др.)
- **Эффект выхода** — анимация исчезновения клипа (Fade Out, Zoom Out и др.)
- **Длительность** каждого эффекта в секундах

### Шаг 8. Добавление субтитров

1. Нажмите **+ Субтитр** на панели инструментов
2. Субтитр появится в начале таймлайна на дорожке субтитров
3. **Перетащите** субтитр по дорожке, чтобы изменить начало
4. **Потяните правый край** для изменения длительности
5. Выберите субтитр и перейдите на вкладку **Субтитры** в панели свойств:
   - Введите текст
   - Настройте шрифт, размер, цвет, выравнивание
   - Выберите анимацию
6. **Позиционирование**: перетащите субтитр прямо в области предпросмотра мышью
7. **Изменение размера**: используйте маркеры по краям субтитра в области предпросмотра

### Шаг 9. Настройка аудио

1. После добавления аудиофайла выберите его на аудиодорожке
2. В панели свойств на вкладке **Аудио** настройте:
   - Громкость (0–200%)
   - Скорость воспроизведения
   - Fade In / Fade Out
   - Смещение по времени (задержка начала)
   - Обрезку начала файла
   - Звуковые эффекты
3. Чтобы добавить звуковой эффект, нажмите **+ Эффект** на панели свойств аудиодорожки

### Шаг 10. Добавление PIP (картинка в картинке)

1. Нажмите **+ PIP** на панели инструментов
2. В появившемся диалоге выберите файл (изображение или видео)
3. PIP-слой появится на дорожке PIP в таймлайне
4. Выберите PIP-слой, перейдите на вкладку **PIP** в панели свойств:
   - Задайте время начала и конца отображения
   - Настройте позицию (X%, Y%), размер (W%, H%)
   - Задайте прозрачность

### Шаг 11. Предпросмотр

- Нажмите **▶** (Пробел) для воспроизведения
- Перетащите **ползунок времени** под предпросмотром для перемотки
- Используйте **Ctrl + колесо мыши** на предпросмотре для масштабирования

### Шаг 12. Сохранение проекта

- **Сохранить** — сохраняет проект на сервере (в JSON)
- **Сохранить как .project** — создаёт ZIP-архив с JSON и всеми медиафайлами; можно скачать и перенести на другой компьютер
- **Открыть .project** — загружает ранее сохранённый архив

### Шаг 13. Экспорт

1. Нажмите кнопку **Экспорт**
2. Настройте параметры во всплывающем окне (качество, разрешение, кодек, формат, аудио)
3. Нажмите **✓ Применить**
4. Прогресс экспорта отображается в реальном времени
5. По завершении — кнопка скачивания готового файла

## Все переходы

Переходы применяются к началу клипа и задаются в панели свойств на вкладке **Переход**.

| Название | Описание |
|----------|----------|
| **Нет** | Прямая склейка без перехода |
| **Fade** | Плавное перекрёстное затухание через прозрачность |
| **Cross Fade** | Альтернативное название Fade (использует тот же алгоритм FFmpeg) |
| **Dissolve** | Растворение: случайные пиксели постепенно меняются местами |
| **Fade Black** | Затухание через чёрный: предыдущий клип гаснет до чёрного, следующий появляется из чёрного |
| **Fade White** | Затухание через белый |
| **Slide Left** | Новый клип въезжает слева, вытесняя предыдущий вправо |
| **Slide Right** | Новый клип въезжает справа |
| **Slide Up** | Новый клип въезжает сверху |
| **Slide Down** | Новый клип въезжает снизу |
| **Wipe Left** | Стирание: новый клип открывается полосой, движущейся влево |
| **Wipe Right** | Стирание вправо |
| **Wipe Up** | Стирание вверх |
| **Wipe Down** | Стирание вниз |
| **Zoom In** | Новый клип появляется из центра, нарастая до полного размера |
| **Pixelize** | Изображение разбивается на крупные пиксели и собирается обратно |
| **Blur** | Горизонтальное размытие в точке перехода |
| **Circle** | Круговое открытие (новый клип проявляется из расширяющегося круга в центре) |
| **Radial** | Радиальное разворачивание, как страница книги |
| **Fade Grays** | Переход через обесцвечивание: оба клипа ненадолго становятся серыми |
| **H Slice** | Горизонтальная нарезка полосами с чередованием |
| **V Slice** | Вертикальная нарезка полосами |

**Советы по переходам:**
- Короткие переходы (0.3–0.5 с) — для динамичных роликов
- Длинные (1.0–1.5 с) — для лирических и документальных видео
- Fade Black отлично разделяет смысловые блоки
- Slide Left/Right работают хорошо при горизонтальном движении в кадре

## Все визуальные эффекты

Эффекты применяются к каждому клипу индивидуально через вкладку **Эффекты** в панели свойств.

### Числовые эффекты (ползунок)

| Эффект | Диапазон | Описание | FFmpeg-фильтр |
|--------|----------|----------|---------------|
| **Яркость** | −100 … +100 | Отрицательные значения — затемнение, положительные — осветление | `eq=brightness=<v/100>` |
| **Контраст** | −100 … +100 | Усиление или ослабление разницы между тёмными и светлыми областями | `eq=contrast=<1+v/100>` |
| **Насыщение** | −100 … +100 | −100 = полностью обесцвеченное, +100 = максимально яркие цвета | `eq=saturation=<1+v/100>` |
| **Размытие** | 0 … 20 | Гауссово размытие. 0 = нет, 5 = заметное, 20 = сильное | `gblur=sigma=<v>` |
| **Резкость** | 0 … 50 | Повышение чёткости краёв через нерезкое маскирование | `unsharp=luma_amount=<v/10>` |
| **Зернистость** | 0 … 50 | Добавление шума — имитация плёночного зерна | `noise=alls=<v>:allf=t+u` |

### Тумблерные эффекты (вкл/выкл)

| Эффект | Описание | FFmpeg-фильтр |
|--------|----------|---------------|
| **Ч/Б** | Перевод в оттенки серого | `hue=s=0` |
| **Сепия** | Коричневатый тон «под старину» | `colorchannelmixer=...` |
| **Виньетка** | Плавное затемнение по краям кадра | `vignette=PI/4` |
| **Инверсия** | Инвертирование всех цветов (негатив) | `negate` |

### Эффекты входа клипа (Start Effect)

Применяются к началу каждого клипа. Задаются в панели свойств → вкладка **Слайд** → раздел «Эффект входа».

| Эффект | Описание |
|--------|----------|
| **Нет** | Клип появляется мгновенно |
| **Fade In** | Клип плавно появляется из прозрачности |
| **Zoom In** | Клип масштабируется от 50% до 100% размера с fade-in |
| **Zoom Out** | Клип масштабируется от 150% до 100% с fade-in |
| **Slide from Left** | Клип въезжает слева |
| **Slide from Right** | Клип въезжает справа |
| **Slide from Top** | Клип въезжает сверху |
| **Slide from Bottom** | Клип въезжает снизу |

### Эффекты выхода клипа (End Effect)

| Эффект | Описание |
|--------|----------|
| **Нет** | Клип обрывается мгновенно |
| **Fade Out** | Клип плавно исчезает |
| **Zoom In** | В конце клип масштабируется от 100% до 150% с fade-out |
| **Zoom Out** | В конце клип уменьшается от 100% до 50% с fade-out |
| **Slide to Left** | Клип уезжает влево |
| **Slide to Right** | Клип уезжает вправо |
| **Slide to Top** | Клип уезжает вверх |
| **Slide to Bottom** | Клип уезжает вниз |

## Анимации субтитров

Задаются в панели свойств при выбранном субтитре → поле «Анимация».

| Анимация | Описание |
|----------|----------|
| **none** | Мгновенное появление без анимации |
| **fade-in** | Плавное появление: субтитр постепенно проявляется из прозрачности |
| **fade-out** | Плавное исчезновение в конце |
| **slide-up** | Субтитр въезжает снизу вверх с fade |
| **slide-down** | Субтитр въезжает сверху вниз с fade |
| **typewriter** | Эффект печатной машинки — субтитр появляется быстро с fade-in |
| **zoom-in** | Субтитр масштабируется от точки до нормального размера |

**Настройка длительности анимации:** поле «Длит. анимации» (секунды). Рекомендуется 0.3–0.8 с.

## Звуковые эффекты для аудиодорожек

Добавляются в панели свойств аудиодорожки → кнопка **+ Эффект**.

| Эффект | Параметры | Описание |
|--------|-----------|----------|
| **Echo** | delay (мс), decay (0–1) | Эхо с задержкой и затуханием |
| **Reverb** | delay (мс), decay (0–1) | Реверберация — имитация акустики помещения |
| **Bass Boost** | gain (дБ) | Усиление низких частот (~60 Гц) |
| **Treble** | gain (дБ) | Усиление высоких частот (~8000 Гц) |
| **Compressor** | ratio | Динамический компрессор — выравнивает уровни громкости |
| **Phone** | — | Имитация телефонного звука (300–3400 Гц) |
| **Radio** | — | Имитация радио (200–3000 Гц) |
| **Low Pass** | freq (Гц) | Фильтр нижних частот — срезает всё выше заданной частоты |
| **High Pass** | freq (Гц) | Фильтр верхних частот — срезает всё ниже заданной частоты |
| **Chorus** | — | Хор — имитация нескольких голосов |
| **Flanger** | — | Флэнджер — волнообразный фазовый эффект |
| **Distortion** | level | Дисторшн — перегруз для гитарного звука |
| **Noise Gate** | — | Шумоподавление через спектральный анализ |
| **Pitch Shift** | semitones | Сдвиг тона на указанное количество полутонов (вверх или вниз) |

На одну дорожку можно добавить несколько эффектов — они применяются последовательно в порядке добавления.

## Форматы и кодеки для экспорта

### Выбор формата

| Задача | Рекомендуемый формат | Кодек |
|--------|---------------------|-------|
| Для большинства задач | **MP4** | H.264 |
| Загрузка на YouTube | **MP4** | H.264, 1080p, 30fps, CRF 18 |
| YouTube Shorts / TikTok / Reels | **MP4** | H.264, 1080×1920 (9:16) |
| Редактирование в видеоредакторе | **MOV** | ProRes |
| Веб-публикация | **WebM** | VP9 |
| Совместимость со старыми устройствами | **AVI** или **MP4** | MPEG-4 или H.264 |
| Анимированная иллюстрация | **GIF** | — (автоматически) |
| Архивное хранение без потерь | **MKV** | H.264 (CRF 0 = lossless) |
| Только аудио | **MP3** / **WAV** / **FLAC** | — |

### Выбор кодека

| Кодек | Когда использовать |
|-------|-------------------|
| **H.264** (libx264) | Для всего. Максимальная совместимость — играет на любом устройстве. По умолчанию. |
| **H.265** (libx265) | Когда важен маленький размер при высоком качестве. Требует современное устройство. |
| **VP9** | Для WebM-файлов и загрузки на YouTube напрямую. |
| **AV1** | Максимальная эффективность, но кодирование очень медленное. Для архивов. |
| **ProRes** | Для дальнейшего монтажа в профессиональных программах (DaVinci Resolve, Premiere). |
| **MPEG-4** | Для совместимости с очень старыми медиаплеерами и телевизорами. |

### Уровни качества (CRF)

CRF (Constant Rate Factor) — чем меньше значение, тем лучше качество и больше размер файла.

| Уровень | CRF | Примерный размер | Скорость кодирования | Применение |
|---------|-----|-----------------|---------------------|------------|
| Очень низкое | 35 | ~0.5 МБ/мин | Очень быстро | Только для тестов и превью |
| Низкое | 28 | ~1.5 МБ/мин | Быстро | Ограниченный объём хранилища |
| **Среднее** | **22** | **~4 МБ/мин** | **Быстро** | **По умолчанию. Хороший баланс** |
| Высокое | 18 | ~8 МБ/мин | Умеренно | YouTube, Instagram, TikTok |
| Очень высокое | 14 | ~15 МБ/мин | Медленно | Профессиональные проекты |
| Максимальное | 8 | ~30 МБ/мин | Медленно | Максимальное качество |
| Без потерь | 0 | ~100+ МБ/мин | Очень медленно | Архивное хранение |

### Готовые пресеты

| Пресет | Качество | Кодек | FPS | Аудио | Применение |
|--------|----------|-------|-----|-------|------------|
| Максимальная скорость | Очень низкое (CRF 35) | H.264 | 30 | 128k | Быстрые тесты |
| Оптимальное | Среднее (CRF 22) | Авто | 30 | 192k | Общее использование |
| Максимальное качество | Очень высокое (CRF 14) | H.264 | 30 | 320k | Архив и мастер |
| Минимальный размер | Низкое (CRF 28) | H.264 | 24 | 128k | Мессенджеры |
| Для социальных сетей | Высокое (CRF 18) | H.264 | 30 | 192k | YouTube, TikTok, Instagram |

### Аудио в экспорте

| Кодек | Битрейт | Применение |
|-------|---------|------------|
| **AAC** (рекомендуется) | 192k | Для MP4, MOV, MKV — лучшая совместимость |
| MP3 | 128–320k | Когда нужна совместимость с MP3-плеерами |

## Советы и хитрости

### Быстрая работа

- **Массовое изменение длительности**: введите нужное значение в поле «Длит. по умолч.» и нажмите «Применить ко всем» — все изображения получат одинаковую длительность
- **Дублирование клипа**: в будущих версиях; пока — добавьте тот же файл повторно
- **Скрыть дорожку**: на текущий момент нет кнопки «Mute» — уберите аудиодорожку перед экспортом
- **Навигация по таймлайну**: прокрутка колесом мыши прокручивает таймлайн горизонтально
- **Точное позиционирование субтитра**: введите координаты X% и Y% вручную в полях панели свойств

### Работа с аудио

- **Синхронизация**: используйте поле «Смещение» для задержки начала аудиодорожки
- **Fade In/Out**: избегайте резких обрывов — всегда добавляйте хотя бы 0.2–0.5 с fade out
- **Несколько музыкальных треков**: добавьте несколько аудиодорожек — они смешиваются автоматически через amix
- **Аудио из видео**: если в видеоклипе есть звук, извлеките его кнопкой «Извлечь аудио» в панели свойств видеоклипа

### Работа с субтитрами

- **Перемещение**: сначала перетащите субтитр в нужное место на таймлайне, потом уточните позицию мышью в области предпросмотра
- **Стиль**: настройте один субтитр «идеально», затем копируйте настройки вручную на другие — в будущих версиях будет стиль по умолчанию
- **Ширина**: потяните маркер на правом краю субтитра в области предпросмотра
- **Вертикальный отступ от края**: параметр Y% = 5 ставит субтитр почти у верхнего края, Y% = 88 — у нижнего

### Производительность

- **Предпросмотр**: воспроизведение в редакторе использует браузерное декодирование без обработки эффектов в реальном времени — финальное качество в экспортированном файле всегда выше
- **Большие проекты**: при многих клипах с HD-видео предпросмотр может тормозить — это нормально; экспорт выполняется серверным FFmpeg и не зависит от браузера
- **GIF**: GIF-экспорт ограничен 15 fps и не поддерживает аудио

### Шаблоны

- **Создание**: настройте проект полностью → нажмите «Сохранить как шаблон» в панели свойств проекта
- **Применение**: откройте список шаблонов, нажмите «Применить» — откроется диалог, где можно перетащить свои медиафайлы поверх слотов шаблона
- **Передача**: шаблоны хранятся на сервере в формате JSON; при необходимости можно скопировать файл вручную

### Экспорт

- **Предварительный просмотр размера**: сводка параметров отображается в нижней части окна экспорта до нажатия «Применить»
- **GIF**: уменьшите разрешение для уменьшения размера файла GIF (360p или 480p вместо 1080p)
- **Аудио без видео**: если нужен только звук, выберите формат MP3 / WAV / FLAC / OGG на вкладке «Формат» в окне экспорта — отдельная кнопка «Экспорт аудио» появится автоматически
- **ProRes**: используйте QuickTime (.mov) — ProRes в MP4-контейнере поддерживается не везде

## Формат файла проекта (.project)

`.project` — это обычный ZIP-архив с именем файла `.project`. Его можно открыть любым ZIP-архиватором.

**Структура архива:**
```
project.json          — JSON с описанием всего проекта
media/
  images/             — изображения (UUID-имена, напр. a3f7b2c1.jpg)
  clips/              — видеоклипы
  audio/              — аудиофайлы
  thumbs/             — миниатюры видеоклипов
```

**Схема project.json (основные поля):**
```json
{
  "id": "abc123",
  "name": "Мой проект",
  "created_at": "2026-01-15T10:00:00",
  "updated_at": "2026-01-15T11:30:00",
  "slides": [
    {
      "id": "s1",
      "type": "image",
      "file": "a3f7b2c1.jpg",
      "url": "/api/imgvid/images/a3f7b2c1.jpg",
      "original": "photo.jpg",
      "duration": 5.0,
      "transition": {"type": "fade", "duration": 0.5},
      "effects": [{"type": "brightness", "value": 10}],
      "startEffect": {"type": "fade-in", "duration": 0.5},
      "endEffect": {"type": "none", "duration": 0.5}
    }
  ],
  "audio": [
    {
      "id": "a1",
      "file": "e4f5a6b7.mp3",
      "volume": 1.0,
      "speed": 1.0,
      "startOffset": 0,
      "fadeIn": 0.5,
      "fadeOut": 1.0,
      "soundEffects": []
    }

      "id": "sub1",

      "y": 88,

```

**Передача проекта на другой компьютер:**
1. Нажмите «Сохранить как .project»
2. Скачайте файл
3. На другом компьютере нажмите «Открыть .project» и выберите файл — все медиафайлы будут автоматически загружены на сервер
